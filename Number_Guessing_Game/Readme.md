Guessing Game (Python)

A simple number guessing game built using Python.

This beginner-friendly project generates a random number between 1 and 20, and the player tries to guess it. The program gives hints after each guess and counts the number of attempts.

 Features

Generates a random number between 1 and 20

Tells the player if the guess is:

Too high

Too low

Validates input (prevents crashes if text is entered)

Counts the number of attempts

Displays total attempts when the correct number is guessed

 Technologies Used

Python 3

random module (built-in Python module)
