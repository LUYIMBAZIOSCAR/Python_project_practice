Smart Grade Evaluator

The Smart Grade Evaluator is a simple Python program that calculates a student’s grade based on their score.

The program asks the user to enter a score (0–100) and then displays the corresponding letter grade.

 How It Works

The user enters a numeric score.

The program checks if the score is valid (between 0 and 100).

Based on the score, a grade is assigned using conditional statements.

 Grading Scale
Score Range	Grade
90 – 100	A
80 – 89	B
70 – 79	C
60 – 69	D
Below 60	F
